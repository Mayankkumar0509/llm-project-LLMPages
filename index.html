Welcome to My GitHub Pages Project

This project includes various files demonstrating a single-page application hosted on GitHub Pages.

  • ashravan.txt - A short story about Ashravan after being restored by Shai.
  • dilemma.json - A scenario involving an autonomous vehicle making ethical decisions.
  • about.md - A brief description of myself in three words.
  • pelican.svg - An SVG image of a pelican riding a bicycle.
  • restaurant.json - A recommendation for a restaurant in Bangalore.
  • prediction.json - A forecast for the Fed Funds rate by December 2025.
  • LICENSE - The MIT License for this project.
  • uid.txt - The unique identifier for the project.
